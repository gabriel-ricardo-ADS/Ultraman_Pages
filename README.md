# Ultraman Pages

A simple multi-page website built with HTML and CSS, inspired by the Ultraman universe.

---

## About the Project

This project was created as part of my early journey in web development, focusing on building a structured and visually organized website using only HTML and CSS.

It demonstrates fundamental concepts such as page structure, navigation, and media integration.

---

## Features

* Multi-page navigation between characters
* Themed content based on Ultraman series
* Image and audio integration
* Organized folder structure
* Basic responsive layout

---

## Technologies Used

* HTML5
* CSS3

---
